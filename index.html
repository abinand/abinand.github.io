Moving Target Pursuit Algorithm

This is an algorithm for provisioning a game agent to pursue moving targets efficiently in a dynamic game world. It is based on an adaptive A* search technique called "Tracking Automatic Optimization". The project was created as part of my engineering coursework and was originally implemented in XNA. The live demo presented in the link below is a re-written HTML5 version of the base algorithm.

Shader Lighting

Demonstrates a fragment shader used to render a diffuse light on a normal map. The surface plane and the scene is created using the three.js library. The normal map is courtesy of Deviant Art. The light color varies depending on the pixel position on the screen.

Quaternion Slerp

A simple WebGL demo to illustrate how Quaternions can be used for interpolating rotations. The desired final orientation is input as euler angles. The angles are converted to a quaternion representation and used for spherical linear interpolation (slerp) from the inital orientation. The actual representations are matrices in WebGL of course but the rotation animations are easier with quaternions and do not have gimbal lock problems as we find in euler angles. The source code is available on github

Tesselation and Twist

A WebGL demo on twisting of primitive shapes by displacing vertices proportional to the distance from the shape's center. This was one of the assignments in an online graphics course. It also shows how the appearance is affected in various levels of tesselation. The effect can be observed for different shapes and fractal geometries.
